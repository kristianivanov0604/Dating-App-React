# Routes - User

##### File Name:  user.js
##### File Path:   /server/routes/user.js

##### Contains following routes:  
  * register,
  * getNearbyUsers,
  * saveVote and
  * createMatches.

##### The functionalities for the routes above are present in following controllers:
  * user Controller,
  * getNearbyUsers Controller,
  * saveVote Controller and
  * match Controller.
